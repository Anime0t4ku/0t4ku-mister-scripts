# MiSTer Scripts

A collection of useful scripts for the MiSTer FPGA platform.

These scripts are designed to improve usability, automate tasks, and add extra functionality to your MiSTer setup. They are intended to be placed in the `/media/fat/Scripts` directory and can be run directly from the MiSTer Scripts menu.

## Installation

**Option 1: Manual**
1. Download or clone this repository.
2. Copy the desired script(s) to `/media/fat/Scripts/`.
3. Run them from the MiSTer Scripts menu.

**Option 2: MiSTer Companion**
- Install and manage scripts directly using MiSTer Companion.

## Notes

- Some scripts may require an internet connection.
- Always review scripts before running them.
- Use at your own risk.
